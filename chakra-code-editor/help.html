Chakra Editor।

কীভাবে শুরু করবেন?

  1. বামদিকের মেনু থেকে টপিক সিলেক্ট করুন।
  2. কোডগুলো মনোযোগ দিয়ে পড়ুন।
  3. Chakra Editor-এ গিয়ে নিজে নিজে টাইপ করে Run বাটনে ক্লিক করুন!

একটি ছোট্ট সিক্রেট!

পৃথিবীর সেরা কোডাররাও সব কোড মুখস্থ করে না! তারা শুধু জানে "কোথায় কী খুঁজতে হয়" এবং "কীভাবে জোড়া লাগাতে হয়"। তাই কোনো কোড মুখস্থ করার চেষ্টা করবেন না। শুধু বুঝুন লজিকটা কীভাবে কাজ করছে। আর আটকে গেলে গুগল বা మనদের এডিটরের Explain বাটন তো আছেই!

টোকেন (Token) নিয়ে সাবধানতা!

আপনার গিটহাবের PAT Token (ghp_...) হলো আপনার এটিএম কার্ডের পিনের মতো! এটি কখনোই পাবলিক চ্যাটে, হোয়াটসঅ্যাপ গ্রুপে বা অন্য কারও সাথে শেয়ার করবেন না। এটি কারও হাতে পড়লে সে আপনার গিটহাবের সব প্রজেক্ট ডিলিট করে দিতে পারে। এটি শুধু আপনার Chakra Editor এর সেটিংসে বসানোর জন্যই ব্যবহার করবেন।

আপনার সাইট লাইভ করুন (GitHub Pages)

আপনি যখন আপনার index.html ফাইলটি গিটহাবে Push করবেন, তখন গিটহাব ওয়েবসাইটে গিয়ে আপনার রেপোর Settings > Pages অপশনে যান। সেখান থেকে Source হিসেবে "main" বা "master" ব্রাঞ্চ সিলেক্ট করে সেভ করুন। ব্যস! ৫ মিনিট পর গিটহাব আপনাকে একটি লিংক দেবে, যা দিয়ে সারা পৃথিবীর মানুষ আপনার বানানো ওয়েবসাইট দেখতে পারবে!

Chakra Editor কীভাবে চালাবেন?

১. নতুন প্রজেক্ট এবং ফাইল তৈরি

এডিটরের উপরে থাকা [+] আইকনে ক্লিক করে নতুন প্রজেক্ট বানান। আর বামদিকের প্যানেল থেকে File Plus আইকনে ক্লিক করে index.html, style.css বা app.js ফাইল তৈরি করতে পারবেন।

২. লাইভ প্রিভিউ (Run)

আপনার লেখা কোড কেমন দেখতে হলো, তা জানতে উপরের সবুজ রঙের ▶ Run বাটনে ক্লিক করুন। সাথে সাথে একটি লাইভ উইন্ডো খুলবে যেখানে আপনার ওয়েবসাইট দেখা যাবে!

৩. জাদুর কাঠি: Explain Code 🪄

কোথাও আটকে গেছেন? কোড বুঝতে পারছেন না? জাস্ট উপরের হলুদ রঙের Explain বাটনে ক্লিক করুন। আমাদের এডিটর আপনাকে বাংলায় সহজ করে বুঝিয়ে দেবে ওই ফাইলে কী লেখা আছে!

৪. ভুল হলে কী করবেন? (Undo/Redo)

ভুল করে কোড মুছে ফেললে ভয়ের কিছু নেই! উপরের বাঁকানো তীর (↩) আইকনে ক্লিক করলেই মুছে যাওয়া কোড আবার ফেরত চলে আসবে।

মোবাইল টিপস (Mobile Pro Tip)

আপনি যদি মোবাইল থেকে কোডিং করেন, তবে অনেক বড় কোড পেস্ট করার পর ঘাবড়ে যাবেন না। আমাদের এডিটরটি খুব স্মার্ট! এটি Word Wrap করে আপনার স্ক্রিনের মাপে কোডকে নিচে নিচে সাজিয়ে দেবে, যাতে আপনাকে ডানে-বামে স্ক্রল করতে না হয়। আর যেকোনো লেখাকে খুঁজতে ওপরের সার্চ আইকনটি ব্যবহার করতে ভুলবেন না।

গিটহাব (GitHub) মাস্টারগাইড

গিটহাব হলো কোডারদের জন্য গুগল ড্রাইভ বা গেমের সেভ পয়েন্ট! এখানে কোড রাখলে তা সারাজীবন সেভ থাকবে।

ধাপ ১: Token (PAT) তৈরি করা (খুব জরুরি!)

  1. এই লিংকে ক্লিক করে গিটহাবে লগইন করুন।
  2. Generate new token (classic) এ ক্লিক করুন।
  3. নাম দিন (যেমন: Chakra Editor)। মেয়াদ দিন (No expiration বা 90 days)।
  4. নিচের লিস্ট থেকে শুধু repo লেখা ঘরটিতে টিক চিহ্ন (✔) দিন।
  5. একদম নিচে গিয়ে Generate Token এ ক্লিক করুন।
  6. যে লম্বা কোডটি (ghp_...) আসবে, সেটি কপি করে যত্ন করে রেখে দিন।

ধাপ ২: Chakra Editor-এ গিটহাব কানেক্ট করা

  • এডিটরের উপরের গিটহাব আইকনে ক্লিক করুন।
  • আপনার কপি করা Token টি বসান।
  • আপনার গিটহাব ইউজারনেম এবং নতুন প্রজেক্টের নাম দিন (যেমন: rahul/my-first-site)।
  • নতুন হলে Create Repo তে ক্লিক করুন, অথবা Save Settings এ ক্লিক করুন।

Push (কোড আপলোড করা)

আপনি এডিটরে যা লিখেছেন, তা গিটহাবে পাঠাতে Commit Message (যেমন: Added a button) লিখে Push Current বাটনে ক্লিক করুন।

Pull (কোড ডাউনলোড করা)

গিটহাব থেকে কোনো ফাইল এডিটরে আনতে ফাইলের নাম (যেমন: index.html) লিখে Pull Files এ ক্লিক করুন।

HTML এর হাতেখড়ি

HTML হলো ওয়েবসাইটের কঙ্কাল। এটি দিয়ে আমরা ওয়েবসাইটকে বলি কোথায় লেখা থাকবে আর কোথায় ছবি থাকবে।

বেসিক স্ট্রাকচার:

<!DOCTYPE html> <html> <head> <title>আমার প্রথম সাইট</title> </head> <body> <h1>হ্যালো কোডার!</h1> <p>আমি হেসে হেসে কোডিং শিখছি।</p> </body> </html>

গুরুত্বপূর্ণ কিছু ট্যাগ:

  • <h1> থেকে <h6>: বড় হেডিং বা টাইটেল দেওয়ার জন্য।
  • <p>: সাধারণ প্যারাগ্রাফ বা টেক্সট লেখার জন্য।
  • <button>ক্লিক করো</button>: ক্লিক করার বাটন বানাতে।
  • <img src="ছবির-লিংক.jpg">: ওয়েবসাইটে ছবি দেখাতে।

Tailwind CSS এর ম্যাজিক

আগে ডিজাইন করার জন্য আলাদা CSS ফাইল লিখতে হতো। Tailwind এর মাধ্যমে আমরা সরাসরি HTML এর ভেতরেই ক্লাসের নাম লিখে ডিজাইন করতে পারি!

কীভাবে কানেক্ট করবেন?

আপনার HTML ফাইলের `<head>` এর ভেতর এই লাইনটি বসিয়ে দিন:

<script src="https://cdn.tailwindcss.com"></script>

ম্যাজিক ক্লাসের উদাহরণ:

<!-- একটি সুন্দর নীল রঙের বাটন --> <button class="bg-blue-500 text-white font-bold py-2 px-4 rounded hover:bg-blue-700"> আমাকে ক্লিক করো </button> <!-- একটি সুন্দর কার্ড --> <div class="p-6 max-w-sm bg-white rounded-xl shadow-lg flex items-center space-x-4"> <div class="text-xl font-medium text-black">Tailwind খুব সোজা!</div> </div>
  • bg-red-500: ব্যাকগ্রাউন্ড লাল করবে।
  • text-white: লেখার রং সাদা করবে।
  • p-4: চারপাশে প্যাডিং (জায়গা) ছাড়বে।
  • rounded-lg: কোণগুলো গোল করে দেবে।

JavaScript এর খেল

HTML কঙ্কাল বানায়, CSS তাকে সুন্দর করে, আর JavaScript তাকে প্রাণ দেয় (যেমন: বাটনে ক্লিক করলে কিছু হওয়া)। এটি `app.js` ফাইলে বা HTML এর `<script>` ট্যাগে লিখতে হয়।

১. স্ক্রিনে মেসেজ দেখানো (Alert & Console)

// ব্রাউজারে পপ-আপ মেসেজ দেখাবে alert("ওয়েবসাইটে স্বাগতম!"); // ভেতরে ভেতরে (Console-এ) মেসেজ প্রিন্ট করবে console.log("জাভাস্ক্রিপ্ট কাজ করছে!");

২. ভেরিয়েবল (বাক্স) তৈরি করা

let myName = "রাহুল"; let age = 20; console.log("আমার নাম " + myName + " এবং বয়স " + age);
